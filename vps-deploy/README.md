# Развертывание на VPS сервере Jino

## Подготовка к развертыванию

Эта папка содержит файлы и инструкции для развертывания Telegram Shop на VPS сервере Jino.

## Структура файлов

- `deploy.sh` - Скрипт автоматического развертывания
- `nginx.conf` - Конфигурация Nginx
- `pm2.config.js` - Конфигурация PM2 для управления процессом
- `setup.sh` - Скрипт первоначальной настройки сервера
- `env.example` - Пример файла с переменными окружения

## Быстрый старт

1. Скопируйте все файлы проекта в эту папку или используйте git clone
2. Скопируйте `env.example` в `.env` и настройте переменные окружения
3. Запустите `./setup.sh` для первоначальной настройки
4. Запустите `./deploy.sh` для развертывания

## Требования

- Node.js 18+ 
- Nginx
- PM2
- SQLite3

## Настройка

Скопируйте `env.example` в `.env` и заполните все необходимые переменные:

```bash
cp env.example .env
nano .env
```

## Порты

- Приложение: порт 3000 (внутренний)
- Nginx: порт 80/443 (внешний)
